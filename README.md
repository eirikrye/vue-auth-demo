# vue-auth-demo

## node

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

## python

### Install

```
pip3 install -r server/requirements.txt
```

### Run

```
cd server
waitress-serve --listen=127.0.0.1:8181 api:app
```
